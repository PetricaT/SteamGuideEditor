# Guide editing made easy

EDITOR: [Github Page](https://petricat.github.io/SteamGuideEditor/)

---

This was made out of necessity if anything. I needed a better way of writing steam-specific markdown and have a real time preview. Enter, this tool, which parses it in real time so you can get a feel for your changes while you work.
